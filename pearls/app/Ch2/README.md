# 上位者問題

[Small programming exercises 20](https://www.sciencedirect.com/science/article/pii/0167642388900238)

上位者とは配列内のある要素の右側にありその要素よりも大きい要素のことです。
`i < j & x[i] < x[j]` ならば、 x[j] は x[i] の上位者です。

上位者数とは配列内のある要素の持つ上位者の数のことです。

GENERATINGの上位者数は

```
G E N E R A T I N G
5 6 2 5 1 4 0 1 0 0
```

になります。
